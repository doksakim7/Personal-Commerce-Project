## 프로젝트 소개
이 프로젝트는 내일배움캠프에서 진행한 커머스 과제입니다. 
Java의 객체지향 개념(클래스 분리, 메서드 설계 등)을 연습하기 위해
콘솔 기반 커머스 플랫폼을 구현했습니다.



## 현재 구현 기능
- 콘솔 메뉴에서 번호 입력에 따라 기능이 실행되도록 구현
- 제품 유형(카테고리) 선택 시 해당 제품 목록 출력
- 제품 선택 시 제품 상세 정보 출력
- 구매 수량 입력 시 재고 차감 처리



## 추가 예정 기능
- 장바구니에 상품 추가/삭제/조회 기능
- 장바구니 내 수량 변경 및 재고/수량 검증 강화
- 주문(결제) 처리 흐름 및 주문 내역 관리
- enum, 제네릭, 람다/스트림을 활용한 관리 기능 고도화



## 기술 스택
- Language: Java 
- JDK: openjdk 17.0.17
- Environment: Console Application
